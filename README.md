# pinefinders.github.io

Landing page / app hub for the Pinefinders suite.

Lives at **https://pinefinders.github.io/**

## Apps linked

- [Auction Finds](https://pinefinders.github.io/auction-finds/) — daily EasyLive pine scraper
- [Auction Map](https://pinefinders.github.io/auction-map/) — UK auction house map with route planner

## Adding a new app

1. Open `index.html`
2. Duplicate one of the `.app-card` blocks inside `.grid`
3. Update the `href`, icon, title, description, tag, and `--card-accent` colour
4. Commit + push — GitHub Pages auto-deploys

The "More coming soon" placeholder can be removed once 4+ real apps are listed.
